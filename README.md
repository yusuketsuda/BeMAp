# BeMAp
BeMAp is a platform for mapping and analyzing plasmids carrying antimicrobial resistance genes.

# Features


# Requirements 
* NCBI blast
* Biopython

# Tested environments
* macOS 11.6
* Ubuntu 20.04
* Python 3.7.10, 3.8.5
* makeblastdb 2.5.0, 2.9.0
* biopython 1.77, 1.79

# Installation


# Usage
```bash
python BeMAp.py -d directory_of_genbank_files -i fasta_file_of_target_gene
```

# Options
* -d, --indir 

* -i, --infile

* -o, --out

* -num_process

* -identity
